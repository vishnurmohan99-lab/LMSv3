Chapter 2 · Selectors & Specificity
Page {{ pdfPage }} of 12

CSS selectors define which elements a rule applies to. The cascade resolves conflicts using specificity, computed as a tuple of (inline, IDs, classes/attributes, elements). When two rules have equal specificity, the one declared later wins.

.card {
  padding: 16px;
  border-radius: 12px;
}

Prefer class selectors for reusability; avoid IDs in CSS to keep specificity low and predictable across a growing codebase.
